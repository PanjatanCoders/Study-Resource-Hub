## GitHub Repositories
- [🐍 30 Days Of Python](https://github.com/Asabeneh/30-Days-Of-Python/tree/master) - 30 days of Python programming challenge is a step-by-step guide to learn the Python programming language in 30 days.
- [Playground and Cheatsheet for Learning Python](https://github.com/trekhleb/learn-python/tree/master) - Playground and cheatsheet for learning Python. Collection of Python scripts that are split by topics and contain code examples with explanations.
- [Python-programming-exercises](https://github.com/zhiwehu/Python-programming-exercises/tree/master?tab=readme-ov-file) - 100+ Python challenging programming exercises
- [Python](https://github.com/geekcomputers/Python?tab=readme-ov-file) - A Python script repository is a hands-on toolkit to explore, automate, and learn real-world coding step by step.
- [Project Based Learning](https://github.com/practical-tutorials/project-based-learning?tab=readme-ov-file#python) - A project-based tutorial repository is a powerful learning hub that helps you apply programming concepts—especially in Python—through real-world projects while building a strong portfolio for job readiness.
- [Amazing-Python-Scripts](https://github.com/avinashkranjan/Amazing-Python-Scripts) - A Python script repository is a handy resource full of task automation, web scraping, and project-ready examples—perfect for students exploring coding on their own or scaling up to bigger builds.
- [The Algorithms - Python](https://github.com/TheAlgorithms/Python) - TheAlgorithms/Python is a great open-source resource packed with Python implementations of algorithms and data structures—perfect for grasping the foundations of computer science and sharpening competitive programming skills. Just keep in mind that the code is educational, not optimized for production use.
